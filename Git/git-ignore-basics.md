# Git Ignore - Advanced Guide

`.gitignore` ගොනුවක් භාවිතා කරන්නේ Git විසින් හඹා නොයා යුතු (untracked) ගොනු සහ ෆෝල්ඩර ලැයිස්තුගත කිරීමටයි.

### 1. පොදු භාවිතයන් (Common Patterns)

- `node_modules/` : සම්පූර්ණ ෆෝල්ඩරයක්ම නොසලකා හැරීමට.
- `*.log` : ඕනෑම නමකින් ඇති `.log` extension එක සහිත ගොනු ඉවත් කිරීමට.
- `config.php` : නිශ්චිත එක් ගොනුවක් පමණක් ඉවත් කිරීමට.
- `temp/*.txt` : `temp` ෆෝල්ඩරය ඇතුළත ඇති සියලුම `.txt` ගොනු ඉවත් කිරීමට.

### 2. ව්‍යතිරේකයන් (Exceptions)

යම් ෆෝල්ඩරයක් ignore කර තිබියදී, එහි ඇති එක ගොනුවක් පමණක් track කිරීමට අවශ්‍ය නම් `!` සලකුණ භාවිතා කරයි:

```text
/logs/*
!/logs/important.log
```
